# YangTools Revit - 优化与迭代工作日志

## 2026-06-12 (v2.0)
### 🌟 新增核心功能 (总控批处理与云链接)
1. **云端模型与本地链接自动化**: 
   - 支持自动化检测与加载 ACC / BIM 360 云端 Revit 模型。
   - 支持自动化扫描指定目录下的 DWG 格式图纸并以“原点对原点”方式全自动锁定链接。
2. **高级批量导出矩阵**:
   - **NWC (Navisworks) 优化**: 彻底修复了在 2D 视图下导出报错的问题；新增按照 Revit 图纸/视图集进行批量循环导出功能。
   - **IFC 模块强化**: 支持选择输出版本（IFC 4 / IFC 2x3），同时新增按图纸/视图集筛选的独立循环输出能力。
   - **PDF 动态策略**: 在导出图纸/视图集时，增加“合并为单文件”或“按图纸拆分”配置选项。
   - **CAD 继承输出**: 无缝继承并调用原生 DWG 预设导出配置。
3. **架构与 UI 优化**: 
   - 彻底移除了对旧版 WinForms (FolderBrowserDialog) 的依赖，全面应用现代化的 WPF 和 Microsoft.Win32 弹窗机制，大幅提升稳定性并削减了冗余依赖。
   - 为上述新增加的配置编写了基于 MVVM 模式的独立数据绑定结构，界面响应与后台解耦。

### 🐞 修复与清理
1. 修复了 .csproj 配置文件中 <Using Remove> 导致的潜在 MSBuild 冲突，提升了项目跨版本编译稳定性。
2. 大规模清理了 C# 8.0 nullable 语法带来的编译器警告。

## 2026-06-09 至 2026-06-11 (v1.9)
### ⚡ 深度架构重构与性能优化
1. **WPF / XAML 绑定优化**:
   - 将 DataGrid 绑定重构为双向 MVVM，废弃 SheetsDataGrid_CellEditEnding 旁路代码。
   - 将所有字典存储重构为 ObservableDictionary，彻底解决 UI 假死与数据更新锁死问题。
   - 新增 SharedStyles.xaml 统一样式，处理高并发弹窗渲染。
2. **Revit API 线程安全**:
   - 清除了项目中未在 using 块释放的野生 Transaction。
   - 对非模态窗口增加了 _doc.IsValidObject 守护，防止 Revit 关闭文档后后台崩溃。
   - 实现了 ElementIdExtensions.CreateId(long) 完成了 Revit 2024 / 2025 完整的 64-bit ID 平滑过渡。
3. **性能突破**:
   - 利用 ToElementIds() 代替 LINQ 遍历 FilteredElementCollector，消除大型项目堆内存瓶颈。
   - 在图纸与过滤器相关功能中引入预加载字典缓存，不再需要实时迭代数百个对象。
